# Docusaurus 使用

## 环境安装

您需要提前安装 Npm 和 Yarn 软件。

### 安装 Npm

参考 [Npm 官网](https://nodejs.org/)。

### 安装 Yarn

```bash
npm install -g yarn
```

## 克隆仓库

```bash
git clone git@github.com:wuminlong/wuminlong.github.io.git
```

## 安装依赖

```bash
yarn install
```

## 查看项目

```bash
yarn start
```

## 编译项目

```bash
yarn build
```

编译成功后，查看项目。

```bash
yarn serve
```

## 部署项目

```bash
yarn deploy
```
